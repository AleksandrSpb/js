Добавьте новую переменную celsius и запишите в неё значение, рассчитанное по формуле перевода температуры из градусов по Фаренгейту в градусы Цельсия: (fahrenheit - 32) / 1.8;

let fahrenheit= 451;
let celsius = (fahrenheit - 32) / 1.8;

В прошлой задаче мы использовали переменные fahrenheit и celsius как числа, в этой — будем работать с ними как со строками. Применив переменные и шаблонные строки, выведите в консоль строку: '_fahrenheit_ градуса по Фаренгейту - это _celsius_ градуса по Цельсию.' где _fahrenheit_ и _celsius_ — это значения соответствующих переменных.

Чтобы создать шаблонную строку, заключите текст в обратные кавычки. Затем поместите переменные внутрь шаблонной строки, заключив их в фигурные скобки с долларом: ${имя переменной}: `${fahrenheit} градуса по Фаренгейту - это ...`

let fahrenheit= 451;
let celsius = (fahrenheit - 32) / 1.8;
console.log(`${fahrenheit} градуса по Фаренгейту - это ${celsius} градуса по Цельсию.`);
